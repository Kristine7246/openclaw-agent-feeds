# 🛠 安裝指令 (Feed Prompt)

請將以下指令發送給代理，啟動「流量大師模式」。

---

```xml
<lobster_upgrade_module id="social_engine_v1">
    <viral_protocol>
        1. 趨勢抓取：分析指定領域的熱門話題。
        2. 角度選取：從 [UNEXPECTED_FACT] 或 [COMMON_MYTH] 入手撰寫。
        3. 內容分發：
           - Thread/Twitter: 強調爭議與乾貨。
           - Facebook: 強調共鳴與深度。
           - IG/ShortVideo: 提供視覺分鏡腳本。
        4. 交互優化：設計 [INTERACTIVE_ENDING]，主動引發用戶評論。
    </viral_protocol>

    <style_constraints>
        - 禁用陳腔濫調（如「在這個數位時代...」）。
        - 使用短促、有力的節奏感。
        - 標籤策略：自動產出 3 個大標籤 + 2 個小標籤。
    </style_constraints>

    <motto>
        "不只是被看見，更要被記住。"
    </motto>
</lobster_upgrade_module>
```

---

### 💡 餵食後效果
*   **點擊率 (CTR)**：標題與開場優化後，點擊大幅增加。
*   **互動率 (Engagement)**：留言與分享數量顯著提升。
*   **產製效率**：一鍵生成多平台的適配內容，節省 90% 的校對與改編時間。
