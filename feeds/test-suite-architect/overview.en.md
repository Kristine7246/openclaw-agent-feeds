# 🧪 [Mutation Feed] Test Suite Architect

### 📄 Module Overview
This significantly empowers AI Agents to be more stable, precise, and less prone to guessing in planning, tool usage, response structuring, and result verification. This feed pack (Mutation Feed) sets out to fortify your repository with an **impenetrable defensive testing moat**. It upgrades your workspace into a tyrannical QA Automation Engineer. By wielding a "Guarded Decision Loop," it prevents the agent from slapping together random mock setups. It forces a methodological Deconstruction (Inspect First) of node dependencies and brutal Assertion Verifications (Verify)—permanently incinerating "Flaky Tests" that crumble under the slightest temporal shift.

### ⚙️ Skill Synergy
- **Recommended Skills**: Within-workspace terminal execution (`run_command`), File read/write ops.
- **Synergy Effect**: Transmutes the terminal into an arena of judgment. Once test scripts are generated, the `Execute` echelon commands the terminal to fire the test runner. Should a red error illuminate, the `Validation` loop relentlessly forces structural rewrites, yielding only when absolute green pathways are proven.

### 🚀 Behavior Upgrade Target
1. **Conditional Branches**: Issued a vacant "add tests" mandate absent a framework context, it violently shunts into the Clarification Branch to pin down Jest, Pytest, or Mocha, refusing chaotic installations.
2. **Mandatory Verify**: Post-generation, it forces severe audits ensuring tests do not hallucinate external non-static API availability, aggressively embedding Stubs/Mocks instead.
3. **Check Tooling**: Surveys local package topology (`package.json`, environment vars) verifying test Runner libraries exist before writing.
4. **Pervasive Scope**: Perpetually imposes these directives during non-trivial tasks spanning TDD pipelines, patching coverage blind-spots, and re-engineering brittle monoliths.

- **Recommended**: Gemini 3.0 Pro / GPT-5.3 / Claude Sonnet 4.6
- **Minimum**: Gemini 3.1 Flash / GPT-5.1
