# 👁️ [Training Feed] Visual Auditor

### 📄 Module Overview
This significantly empowers AI Agents to be more stable, precise, and less prone to guessing in planning, tool usage, response structuring, and result verification. This feed pack (Training Feed) is crafted expressly for **pixel-level UI/UX surveillance and frontend replication**. It upgrades your workspace into a hyper-vigilant Design Director. Dictated by the "Guarded Decision Loop," it commands agents to anchor baselines (Assess Design) and simulate extreme viewport destruction (Verify) prior to altering DOM structures, permanently eradicating instances of AI hallucinating phantom CSS coordinates.

### ⚙️ Skill Synergy
- **Recommended Skills**: Within-workspace visual ingestion tools (`read_image`, Screenshot APIs), Frontend coding tools.
- **Synergy Effect**: Melds abstract frontend syntax with concrete multi-modal vision anchors. During the `Verify` cycle, altering CSS triggers an unavoidable self-audit assessing whether the fix mutated adjacent columns into catastrophic layout shifts.

### 🚀 Behavior Upgrade Target
1. **Conditional Branches**: Triggered by massive dissonance between the rendered screenshot and underlying source code architectures, it jumps to the Clarification Branch. It rejects brute-forcing inline CSS, demanding confirmation if the visual target is obsolete.
2. **Mandatory Verify**: Prior to outputting styling code, it forcefully simulates extreme stress parameters: "If the viewport shatters into a mobile aspect ratio (RWD), does this flexbox container hemorrhage its bounds?"
3. **Check Components**: Initially sweeps the project seeking existing Design Systems or utility grids (e.g., Tailwind directives). Hardcoding isolated inline variables is entirely forbidden.
4. **Pervasive Scope**: Perpetually imposes these directives during non-trivial tasks manipulating visual arrays, RWD stabilization, and accessibility (a11y) audits.

### 📋 Recommended Models
- **Recommended**: Gemini 3.0 Pro / GPT-4o (Models armed with high-tier Vision utility)
- **Minimum**: STRICTLY BANS pure text models (Multimodal Vision is absolute).
