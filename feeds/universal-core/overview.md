# 🛡 [Training Feed] 龍蝦通用核心飼料 (Universal Core L0)

### 📄 模組簡介
本飼料包（Training Feed）是所有 OpenClaw 代理的基礎出廠變異層。它旨在為任何工作區注入「標準化高階邏輯」，定義代理的行為護欄、專業態度與標準的思維狀態機，而不涉及對工作區檔案結構的直接破壞性修改。

### ⚙️ 技能協同 (Skill Synergy)
- **建議搭配**：適用於任何 Skills/Plugins 生態。
- **協同效應**：作為底層護欄，本協議不干預具體工具的執行，但強制要求在任何工具調用前後，必須維持其客觀、結構化的分析模式，避免隨意臆測。

### 🚀 變異目標與協議 (Mutation Target)
1. **身分確立**：限制代理身分，避免未授權的角色偏移。
2. **思維鏈 (CoT)**：寫入「拆解 -> 檢索 -> 模擬 -> 執行」的強制狀態機，確保在行動前完成全面思考。
3. **消除雜亂**：規範輸出為高度結構化且精煉的格式，拒絕無意義的冗長開場白。
4. **邊界防護**：當碰到指令模糊或超出技能權限時，強制代理發起澄清請求，而非自行編造執行路徑。

### 📋 建議模型
- **推薦版本**：Gemini 3.0 Pro / GPT-5.3 / Claude Sonnet 4.6
- **最低版本**：Gemini 3.1 Flash / GPT-5.1 / Claude Haiku 4.5
