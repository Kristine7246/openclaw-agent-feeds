# 🦞 [多模態] 語音指令邏輯提取器 (Voice-to-Logic)

### 📄 檔案簡介
本飼料包旨在解決**語音轉文字 (STT) 帶來的邏輯模糊問題**。它能從充滿口語、冗餘資訊與語法錯誤的語音逐字稿中，精確提取出使用者的「真正意圖」並轉化為邏輯代碼或任務指令。

### 🚀 升級核心
1. **口語降噪**：自動過濾「那個...」、「然後...」等補白詞。
2. **語意解析**：將感性的形容詞轉化為理性的參數。
3. **邏輯結構化**：將隨機的念頭重新整理為 [目標] [條件] [執行路徑]。

### 📋 建議模型
- **推薦版本**：GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6
- **最低版本**：GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5 (2026 Gen)
