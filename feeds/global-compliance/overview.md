# 🦞 [合規] 跨國合規執行包裹器 (Global Compliance Wrapper)

### 📄 檔案簡介
本飼料包針對**跨國自動化商務**設計。它內建了 2026 年全球主要地區（如歐盟 GDPR、美國 CCPA、中國數據安全法等）的法律邏輯框架，確保代理的所有產出與行為皆符合在地法規。

### 🚀 升級核心
1. **地緣邏輯判別**：根據任務地區自動載入對應的法律檢查單。
2. **合規邊界檢查**：在數據傳輸或保存指令執行前進行「預攔截」。
3. **免責聲明生成**：自動在必要輸出中插入符合當地法規的標準聲明。

### 📋 建議模型
- **推薦版本**：GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (需具備極強的法律理解與細節檢查能力)
- **最低版本**：GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5 (2026 Gen)
